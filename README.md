# HR Code Onboarding Accelerator
## IBM Bob Dev Day Hackathon 2026

## Overview
A complete HR Employee Management System built to demonstrate 
how IBM Bob can instantly understand, explain, document and 
review any codebase — turning weeks of onboarding into minutes.

## Problem Statement
New developers joining a team spend 2+ weeks reading code 
before they can contribute. This wastes time and slows delivery.

## Solution
IBM Bob reads the entire codebase and instantly:
- Explains what the code does
- Documents the architecture
- Generates tests
- Reviews code quality
All in minutes — not weeks.

## Tech Stack
- Backend: Python FastAPI
- Frontend: HTML CSS JavaScript
- Testing: pytest
- AI: IBM Bob IDE + IBM watsonx.ai Granite model
- Version Control: GitHub

## Modules

### Employee Module (app/employee.py)
Manages employee data with in-memory storage.
- add_employee — adds new employee
- get_employee — retrieves employee by ID
- list_employees — returns all employees
- update_salary — updates employee salary
- calculate_experience_years — calculates years of service
- get_employees_by_department — filters by department

Sample employees:
1. Raj Kumar - Engineering - Rs.75000
2. Priya Sharma - HR - Rs.65000
3. Arun Patel - Finance - Rs.70000
4. Meena Iyer - Engineering - Rs.80000
5. Suresh Babu - Operations - Rs.60000

### Leave Module (app/leave.py)
Handles all leave management with eligibility rules.
- annual leave: 18 days
- sick leave: 10 days
- maternity leave: 182 days
- paternity leave: 5 days

Functions: apply_leave, approve_leave, reject_leave,
get_leave_balance, check_leave_eligibility, get_leave_history

### Payroll Module (app/payroll.py)
Handles salary calculations and payslip generation.

Functions: calculate_monthly_salary, apply_deductions,
calculate_tax, generate_payslip, get_payroll_history,
calculate_bonus

### API Module (app/main.py)
FastAPI application with CORS enabled.

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /health | Health check |

| POST | /employees | Add new employee |
| GET | /employees/{id} | Get employee by ID |
| GET | /leave/{emp_id} | Get leave balance |
| POST | /leave/apply | Apply for leave |
| POST | /leave/approve | Approve leave |
| GET | /payroll/{emp_id} | Get payroll details |
| GET | /payroll/{emp_id}/payslip | Generate payslip |
| GET | / | Frontend dashboard |
| GET | /docs | API documentation |

## How to Run Locally
```bash
# Clone the repository
git clone https://github.com/sworna-vidhya-m/hr-code-onboarding-bob.git
cd hr-code-onboarding-bob

# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the application
uvicorn app.main:app --reload

# Open browser
http://localhost:8000
```

## How to Run Tests
```bash
source .venv/bin/activate
pytest tests/ -v
```

## Business Rules
- Employees need minimum 6 months experience for maternity leave
- Annual leave cannot exceed 18 days per year
- Carry forward maximum 10 days to next year
- Salary on 28th of every month
- Tax calculation based on Indian tax slabs
- Reimbursement must be submitted within 30 days
